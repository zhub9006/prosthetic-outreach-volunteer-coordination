# Prosthetic Outreach Volunteer Coordination

A centralized project to coordinate volunteer tasks across outreach teams for the prosthetic outreach initiative.

## Team Locations
- **Chicago, IL** (Midwest Team)
- **Boston, MA** (Northeast Team)
- **Atlanta, GA** (Southeast Team)

## Quick Links
- [Project Board](https://github.com/orgs/zhub9006/projects)
- [Issue Tracker](https://github.com/zhub9006/prosthetic-outreach-volunteer-coordination/issues)

## Workflows
- **Medical Team**: Coordination with clinical trial sites, patient screening, prosthetic specialist volunteers
- **Logistics Team**: Supply chain, shipping, equipment distribution
- **Community Outreach**: Event planning, local partnerships, awareness campaigns
